# App Store Screenshot Specifications

## Required Devices & Sizes

### iPhone 6.7" (Required)
- Resolution: 1290 x 2796 px
- Devices: iPhone 15 Pro Max, iPhone 14 Pro Max

### iPhone 6.5" (Required)
- Resolution: 1284 x 2778 px OR 1242 x 2688 px
- Devices: iPhone 14 Plus, iPhone 11 Pro Max

### iPhone 5.5" (Optional but recommended)
- Resolution: 1242 x 2208 px
- Devices: iPhone 8 Plus, iPhone 7 Plus

### iPad Pro 12.9" (Required if supporting iPad)
- Resolution: 2048 x 2732 px

6. **Full Codex Reading** — Codex reading page with detailed synthesis
7. **Birth Chart** — Natal chart visualization with planetary positions
8. **Life Current Tracker** — Tracker page showing congruence score over time

## Screenshot Guidelines

- Use real profile data (not placeholder/lorem ipsum)
- Dark background (#1A0E07) with gold accents (#D4A85F)
- No status bar clutter (use clean status bar or hide it)
- Can add minimal text overlays in the app's typography (Space Grotesk / Playfair Display)
- Apple allows device frames — use iPhone 15 Pro frames for premium look

## Tools for Screenshot Generation

- **Xcode Simulator**: Capture from iOS simulator at exact resolutions

- **AppMockUp.io**: Free tool to add device frames and text overlays
