# App Store Review Notes

## For Apple Review Team

Soul Codex is a personality profiling and daily guidance app that synthesizes astrology, numerology, Human Design, and behavioral pattern analysis into a unified soul profile.

### How to Test

1. Open the app — you'll see a landing page
2. Tap "Begin Your Reading" to start the onboarding
3. Enter any name, birth date, birth time, and birth location
4. Answer the behavioral pattern questions (stress patterns, decision style, energy drains)
5. After completing onboarding, you'll receive your Soul Blueprint
6. Navigate through the tabs: Today (daily guidance), Profile (your full profile), Guide (AI chat), Codex (full reading)

### Access Code for Premium Testing

Use access code: **guest123t** on the Profile page to unlock all premium features during review.

### Account Creation

Account creation is optional. The app works fully in anonymous/session mode. Users can optionally create an email/password account to persist their data across devices.

### AI Content

The app uses Google Gemini AI to generate personalized text content (biographies, daily guidance, chat responses). All AI-generated content is personality/lifestyle focused and contains no objectionable material.

### In-App Purchases

Premium features are gated behind:
- Access codes (promotional)
- Apple In-App Purchases (Tiered access)

### No User-Generated Public Content

All content is private to the individual user. There is no social feed, public profiles, or user-to-user communication.
